Dashboard de Inteligência Operacional - 40º BPM/I 🚔📊
Este projeto é um sistema de suporte à decisão desenvolvido para otimizar a análise de estatísticas criminais e indicadores de performance do 40º Batalhão de Polícia Militar do Interior (Votorantim/SP).

O objetivo é transformar dados brutos de planilhas e registros policiais em visualizações dinâmicas que auxiliem no planejamento estratégico e no policiamento orientado pela inteligência.

🚀 Tecnologias Utilizadas
Frontend: React.js + Vite (Interface rápida e reativa)

Backend: Node.js com Express (API de integração)

Banco de Dados: Supabase / PostgreSQL (Armazenamento em nuvem com alta integridade)

Deploy: Vercel (CI/CD integrado ao GitHub)

Estilização: Tailwind CSS (Interface moderna e limpa)

💡 Funcionalidades Principais
Upload Inteligente: Processamento de arquivos CSV com validação de dados.

Sincronização Cloud: Integração em tempo real com banco de dados PostgreSQL via Supabase.

Dashboards Interativos: Gráficos de tendências criminais e indicadores operacionais.

Tratamento de Dados: Lógica de Upsert para evitar duplicidade de registros e garantir que os dados estejam sempre atualizados.

🛠️ Como o projeto foi construído
O projeto utiliza uma arquitetura moderna onde o Frontend se comunica com uma API que gerencia as requisições ao Supabase. A segurança é garantida através de variáveis de ambiente, mantendo as chaves de acesso ao banco de dados protegidas e seguindo as melhores práticas de desenvolvimento seguro.
